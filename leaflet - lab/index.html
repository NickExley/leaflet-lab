Voter Turnout: U.S. House of Representatives (2000 - 2018)

This map shows the voter turnout in each state by percentage of the voting population. Each click of the sequence bar above changes the proportional symbols to reflect the percentage of the voting population of each state who voted in each election year. Map can be adjusted from state view to satellite view via radio buttons in upper right.

Voter turnout data was obtained from www.ballotpedia.org.
